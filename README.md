# 📊 Task 1: Data Cleaning & Wrangling

## 📌 Overview
This project is part of my Data Analytics Internship.  
The objective of this task is to understand raw data and prepare it for analysis by performing data cleaning and transformation.
---
## 🎯 Objective
- To explore and understand the dataset  
- To identify data quality issues  
- To clean and transform the data into a usable format  

---

## 🔍 Steps Performed

### 1. Data Understanding
- Explored dataset structure and columns  
- Identified data types and basic statistics  
### 2. Data Quality Check
- Checked for missing values  
- Identified duplicate records  
- Found inconsistencies and outliers  
### 3. Data Cleaning
- Handled missing values  
- Removed duplicates  
- Corrected inconsistent formats  
### 4. Data Transformation
- Standardized data formats  
- Created new features (if required)  
- Prepared final cleaned dataset  
---
## 🛠️ Tools & Technologies
- Python (Pandas) / SQL  
- Jupyter Notebook / VS Code

## 📊 Outcome
- Successfully converted raw data into a clean and structured dataset  
- Dataset is now ready for further analysis
